Using the Cloudflare API requires authentication so that Cloudflare knows who is making requests and what permissions you have.
Create an API token
to grant access to the API to perform actions.
Note
You can also authenticate with
API keys
, but these keys have
several limitations
that make them less secure than API tokens. Whenever possible, use API tokens to interact with the Cloudflare API.
