Enterprise API Specifications

The NEMESIS engine exposes a high-performance REST API for programmatic deployment of tracing swarms, AI extraction, and graph querying.

POST /api/v1/trace/deploy Deploy OmniChain Agent

Payload:

{
  "seeds": "0x123... \n bc1q...",
  "chain_override": "AUTO",
  "max_depth": 15
}
POST /api/v1/batch/deploy Deploy Multi-Agent Swarm

Payload:

{
  "raw_ledger": "Victim Name: IRGC Case\nSuspect Wallet Addresses: 0x..."
}
GET /api/v1/trace/{trace_id}/graph Extract Aggregated Topology

Returns array of Nodes (with total_inflow/outflow) and Edges for UI rendering.

POST /api/v1/ai/file_intelligence Gemini AI Extraction

Accepts multipart/form-data files. Returns structured JSON containing entities, incidents, and threat profiles extracted from unstructured text.
